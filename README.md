# BOOST Intensive - 30 Landing Page Variations

## 🎨 Проект
30 уникальных вариантов лендинга BOOST intensive в разных визуальных стилях.

## 🚀 Live Demo
**Главная страница**: https://osipov-denis.github.io/boost-landing-page/

На главной странице вы найдете интерактивную навигацию по всем 30 вариантам с возможностью просмотра каждого лендинга одним кликом.

## 📁 Содержание
Этот репозиторий содержит 30 HTML-файлов, каждый из которых представляет собой уникальный визуальный дизайн лендинга курса intensive программирования BOOST.
